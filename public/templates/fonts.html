Rascal Font Book

This page illustrates the Molengo, URW Gothic, DroidSans and VT323 fonts installed on the Rascal. For example, you could use VT323 in your project as follows:

<html>
    <head>
        {% raw %}{% include "include/rascal-head.html" %}{% endraw %}
        <style>
            body { padding: 10px; background-color: #333; }
            .vt323 { font-family: "vt323", sans-serif; font-size: 20px; color: lime; }
        </style>
    </head>
    <body>
        <div class="vt323">Simulation of DEC VT320 Text Terminal</div>
    </body>
</html>
