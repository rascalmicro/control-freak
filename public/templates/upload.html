File Upload

There are several ways you can upload files to Rascal:

scp - secure copy (remote file copy program)
You can use the scp program in a console session on your OS X or Linux desktop to copy files to and from a Rascal: 

scp ./somefile.js root@rascalNN.local:/var/www/public/static/
sftp - secure file transfer protocol
You could use the FireFTP Add-on for the FireFox browser to copy files to and from a Rascal. Set the connection type to SFTP and login as root.
Choose File button
You can use an <input type="file"> button to browse for files and upload them but it is rather ugly. This demo improves the button's appearance by overlaying a standard Rascal button. For safety the demo is restricted to uploading image files to the folder static/uploads. You will need to create this folder with the editor before running the demo. After uploading an image file, you will be able to view it in the editor
Drag and Drop
If you are using an HTML5 compliant browser, you can copy files to Rascal by dragging them from your desktop and dropping them onto your browser. You can use this page to check that your browser has the necessary support. This demo illustrates drag and drop. Like the previous demo, only image files are accepted and folder static/uploads must exist.
Album
The album demo builds on the Drag and Drop demo to provide a simple photo album with a slideshow. Simply drag some pictures onto the frame. Pictures with a 4:3 aspect ratio work best. The folder static/pictures must exist.
Drag and Drop to the Editor
The editor supports dragging of one or more files from your desktop and dropping them onto the file tree. Images, HTML, CSS, JavaScript or Python files can be transferred.
