Known Issues

  1. Doesn't apply Wrap lines or Match brackets preference - fixed
  2. Logs incorrect value when applying Visible tabs - fixed
  3. Restoring default preferences breaks highlight active line - fixed
  4. When line numbers are enabled in solarized themes, gutter theme is not adjusted - fixed
  5. Cursor barely visible in solarized-dark - fixed
  6. After attempting to create a template or folder that already exists, the error message persists even when the dialogue is cancelled - fixed
  7. Button text rendered in serif font in some versions of Linux - fixed
  8. Changing to a supported theme after using an unsupported theme doesn't update non-editor panes (except when changing to blackboard). Workaround is first to select blackboard then another supported theme, or reload editor
  9. Dragging a file from the desktop into an open empty folder leaves the folder able to be deleted unless you move the cursor off it (normally only empty folders can be deleted). Deleting folder also deletes file(s) just dragged into it.
  10. When dragging and dropping files in the file tree, folders sometimes remain highlighted until you move the mouse over them. This has been the case since drag and drop was introduced.
